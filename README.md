Ejercicios-PHP
Repositorio para guardar ejercicios de php en bachillerato 3mc Informatica 

Para ejecutar los archivos deben de tener Docker instalado previamente.
  Una vez ingresado a la carpeta del proyecto escribe el comando "docker compose up" y en el navegador https:/localhost:( el puerto asignado en el docker con "docker ps")
  para el  proyecto login tiene que ir a la carpeta frontend e ir al archivo index.php
